## Задание
В этой задаче ваша цель - добавить скобки к заданному арифметическому
выражению, чтобы максимизировать его значение.

`max(5 − 8 + 7 × 4 − 8 + 9) =?`
## Input / Output 

| Input                                | Output |
|--------------------------------------|--------|
| 1+5                                  | 6      |
| 5-8+7*4-8+9                          | 200    |
